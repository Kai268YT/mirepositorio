## 🚀 Instalación Rápida

```bash
# Clona el repositorio
git clone https://github.com/tuusuario/turepositorio.git

# Navega al directorio
cd turepositorio

# Instala las dependencias
npm install

# Inicia el proyecto
npm start
```

### 📋 Requisitos Previos

- **Node.js** >= 16.0.0
- **npm** >= 8.0.0
- **Git** >= 2.0.0
